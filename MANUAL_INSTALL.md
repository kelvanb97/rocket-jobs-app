# Manual Install

If you prefer to set up the project without an AI coding assistant, follow these steps.

## Prerequisites

- git
- Node.js >= 24
- pnpm >= 10

## Install

Clone the repository `https://github.com/rocket-jobs-ai/rocket-jobs-app.git`

Install dependencies `pnpm install`

Start the dev server `pnpm dev`

## Setup

1. Open <http://localhost:3000/llm-config> to add your Anthropic and/or OpenAI API key and select models for each use case (scoring, resume extraction, cover letters).

2. Open <http://localhost:3000/settings> to configure your profile, scraper sources, and other preferences. You can upload a resume (PDF or DOCX) to auto-fill your profile, work experience, and education.
