# @aja-core/dates

Date and time formatting utilities for human-readable display.

## Exports

- `formatDateWithOrdinal(dateString)` — formats as "January 1st, 2023"
- `timeAgo(isoString)` — relative time like "5min ago", "2d ago"
- `formatInterval(interval)` — converts duration strings to compact format like "2d", "3h"
