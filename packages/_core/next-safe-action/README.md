# @aja-core/next-safe-action

Wrapper around `next-safe-action` providing a configured action client with consistent error handling.

## Exports

- `.` — `actionClient`, `SafeForClientError`, `UnknownServerError`, `DEFAULT_ERROR_MESSAGE`
- `./hooks` — React hooks for consuming server actions

## Dependencies

- `next-safe-action` — type-safe server actions
