# @aja-design/ui

Component library built on Radix UI primitives and styled with Tailwind CSS. Heavily influenced by [shadcn/ui](https://ui.shadcn.com/).

## Features

- 40+ UI components (button, card, dialog, input, toast, etc.)
- Layout primitives (`Flex`, `XStack`, `YStack`)
- Icon set (GitHub, Discord, Google, Lucide icons)
- Lottie animations
- Phone number input, file upload, carousel, command menu
- Global CSS and PostCSS config exports

## Key Dependencies

- `@radix-ui/*` — accessible component primitives
- `tailwindcss` — utility-first CSS
- `lucide-react` — icons
- `sonner` — toast notifications
- `cmdk` — command menu
- `class-variance-authority` — component variants

## Dependencies

- `@aja-core/localstorage` — theme persistence
